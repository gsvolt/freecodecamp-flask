### Basic commands to run to get Flask running

```
# Install virtualenv
sudo python3 -m pip install virtualenv
sudo python3 -m pip3 install virtualenv

# Generate venv folder
python3 -m venv venv

# Activate virtual env
. venv/bin/activate

# Install Flask
pip3 install Flask

# Upgrade pip itself
pip install --upgrade pip

# Run first example
python main.py
```
